Smooth Animations: The bird has fluid animations for jumping and idling, enhancing the visual appeal and gameplay experience.

Character Selection: Players can choose from multiple characters, each with unique animations.

Challenging Gameplay: The game maintains the original's challenging nature, requiring precise timing and quick reflexes to succeed.

C/C++ Projects:

Project 1: Train Ticket Reservation System

Built using C, the Train Ticket Reservation System is a robust application designed to streamline the process of booking and managing train tickets. Here's an overview of its features:

Ticket Management: Users can seamlessly purchase tickets, view available trains, check seat availability, and access their past reservations.

Data File System: Leveraging data files, the application stores crucial information such as train schedules, seat availability, and user reservations. This ensures persistent data storage, allowing users to retrieve their information even after closing the terminal.

Interactive Interface: The user interface facilitates easy navigation, guiding users through ticket booking, seat selection, and reservation management.

Reservation Tracking: Upon booking a ticket, users provide their name for reservation purposes. Each reservation is associated with a unique code, enabling users to later modify or cancel their bookings.

Multiple Train Options: Offering flexibility, the application presents users with a choice of three trains, each with its schedule and seat availability.
